# AI 基础学习路径

---

## 阶段一：入门基础（1-2 周）

**学习内容：**

-   OpenAI SDK 基础
-   API Key 配置与管理
-   基础 API 调用（Chat Completions）
-   参数调优（temperature、max_tokens、top_p）
-   错误处理与重试机制
-   Vercel AI SDK 基础
-   React Hooks 使用（useChat、useCompletion）
-   流式响应处理
-   多提供商切换

**基础项目实践：**

-   简单聊天界面
-   文本补全功能
-   基础提示词工程

**掌握能力：**

-   能够集成 OpenAI API 到 React 项目
-   能够使用 Vercel AI SDK 构建基础 AI 功能
-   理解流式响应与实时更新的实现
-   掌握基础的提示词编写技巧
-   能够处理 API 错误与异常情况

**实践项目：**

-   聊天机器人界面
-   文本生成工具（如代码注释生成）
-   简单的问答系统

**学习资源：**

-   [OpenAI API 官方文档](https://platform.openai.com/docs)
-   [Vercel AI SDK 文档](https://sdk.vercel.ai/docs)
-   项目示例代码

## 阶段二：进阶应用（2-3 周）

**学习内容：**

-   LangChain.js 核心概念
    -   Chains（链式调用）
    -   Prompts（提示词模板）
    -   Memory（对话记忆）
    -   Agents（智能代理）
    -   Tools（工具集成）
-   高级提示词工程
    -   Few-shot Learning
    -   Chain of Thought (CoT)
    -   ReAct 模式
    -   结构化输出
-   数据加载与处理
    -   Document Loaders
    -   Text Splitters
    -   Vector Stores（向量存储）
    -   Embeddings（嵌入向量）
-   工具集成
    -   网络搜索集成
    -   数据库查询
    -   API 调用工具
    -   自定义工具开发

**掌握能力：**

-   能够构建多步骤 AI 工作流
-   能够实现上下文记忆与对话历史管理
-   能够集成外部工具（搜索、数据库等）
-   掌握高级提示词技巧
-   能够构建 RAG（检索增强生成）应用
-   能够开发 AI Agent 应用

**实践项目：**

-   智能客服系统（带知识库检索）
-   数据分析助手（集成 SQL 查询）
-   代码审查助手（多步骤分析）
-   文档问答系统（RAG 架构）

**学习资源：**

-   [LangChain.js 官方文档](https://js.langchain.com/docs/)
-   LangChain 教程
-   开源 LangChain 项目

## 阶段三：浏览器端 AI（2-3 周）

**学习内容：**

-   TensorFlow.js 基础
    -   模型加载与运行
    -   模型转换（Keras → TensorFlow.js）
    -   实时推理
    -   性能优化
-   预训练模型使用
    -   MobileNet（图像分类）
    -   PoseNet（姿态检测）
    -   Speech Commands（语音识别）
    -   Universal Sentence Encoder（文本嵌入）
-   Transformers.js 使用
    -   模型加载与缓存
    -   文本分类、情感分析
    -   命名实体识别（NER）
    -   文本生成（小模型）
-   性能优化
    -   Web Workers 并行处理
    -   模型量化与压缩
    -   缓存策略
    -   渐进式加载

**掌握能力：**

-   能够在浏览器中运行机器学习模型
-   能够处理图像、音频、文本数据
-   能够实现离线 AI 功能
-   掌握前端模型优化技巧
-   能够构建隐私保护的 AI 应用
-   能够实现实时 AI 推理（如摄像头实时识别）

**实践项目：**

-   浏览器端图像分类器
-   实时人脸识别应用
-   离线文本情感分析工具
-   语音命令识别系统
-   浏览器端文档处理工具

**学习资源：**

-   [TensorFlow.js 官方文档](https://www.tensorflow.org/js)
-   [Transformers.js 文档](https://xenova.github.io/transformers.js/)
-   [Hugging Face 模型库](https://huggingface.co/models)

## 学习路径时间表

| 阶段                | 时间   | 难度       | 核心能力         |
| ------------------- | ------ | ---------- | ---------------- |
| 阶段一：入门基础    | 1-2 周 | ⭐⭐       | 基础 AI 集成能力 |
| 阶段二：进阶应用    | 2-3 周 | ⭐⭐⭐     | 复杂 AI 应用开发 |
| 阶段三：浏览器端 AI | 2-3 周 | ⭐⭐⭐     | 离线 AI 应用开发 |
| 阶段四：专业领域    | 3-4 周 | ⭐⭐⭐⭐   | 专业 AI 应用开发 |
| 阶段五：高级架构    | 持续   | ⭐⭐⭐⭐⭐ | 架构设计与优化   |

**总计：8-12 周（基础到中级），高级能力需要持续实践**

---

## 学习建议

-   **循序渐进：** 每个阶段完成后再进入下一阶段
-   **项目驱动：** 每个阶段至少完成 1-2 个实践项目
-   **代码实践：** 多写代码，参考开源项目
-   **社区参与：** 关注 GitHub、Stack Overflow、Discord
-   **持续更新：** AI 技术更新快，保持学习
