## 数组去重

在 JavaScript 中，数组去重是一个常见的操作。以下是几种常用的数组去重方法，每种方法都有其优点和适用场景。

### 方法一：使用 `Set`

`Set` 是 ES6 引入的一种数据结构，它类似于数组，但其成员都是唯一的。利用 `Set` 可以很方便地实现数组去重。

```javascript
const array = [1, 2, 2, 3, 4, 4, 5];
const uniqueArray = [...new Set(array)];
console.log(uniqueArray); // 输出: [1, 2, 3, 4, 5]
```

### 方法二：使用 `filter` 和 `indexOf`

通过 `Array.prototype.filter` 和 `Array.prototype.indexOf` 方法，可以实现数组去重。

```javascript
const array = [1, 2, 2, 3, 4, 4, 5];
const uniqueArray = array.filter((item, index) => array.indexOf(item) === index);
console.log(uniqueArray); // 输出: [1, 2, 3, 4, 5]
```

### 方法三：使用 `reduce` 和 `includes`

通过 `Array.prototype.reduce` 和 `Array.prototype.includes` 方法，也可以实现数组去重。

```javascript
const array = [1, 2, 2, 3, 4, 4, 5];
const uniqueArray = array.reduce(
    (acc /**当前结果 */, item /** 当前元素 */) => {
        if (!acc.includes(item)) {
            acc.push(item);
        }
        return acc;
    },
    [] /** 初始值 */
);
console.log(uniqueArray); // 输出: [1, 2, 3, 4, 5]
```

### 方法四：使用 `forEach` 和对象键值对

通过 `Array.prototype.forEach` 和对象键值对，可以实现数组去重。

```javascript
const array = [1, 2, 2, 3, 4, 4, 5];
const uniqueArray = [];
const seen = {};

array.forEach(item => {
    if (!seen[item]) {
        uniqueArray.push(item);
        seen[item] = true;
    }
});

console.log(uniqueArray); // 输出: [1, 2, 3, 4, 5]
```

### 方法五：使用 Lodash 库

如果你在项目中使用了 Lodash 库，可以使用其 `uniq` 方法来实现数组去重。

```javascript
const _ = require('lodash');
const array = [1, 2, 2, 3, 4, 4, 5];
const uniqueArray = _.uniq(array);
console.log(uniqueArray); // 输出: [1, 2, 3, 4, 5]
```

### 总结

以上方法都可以实现数组去重：

1. **使用 `Set`**：最简洁、性能较好，适用于大多数场景。
2. **使用 `filter` 和 `indexOf`**：代码简洁，适用于小规模数组。
3. **使用 `reduce` 和 `includes`**：适用于需要累积结果的场景。
4. **使用 `forEach` 和对象键值对**：适用于需要手动控制去重逻辑的场景。
5. **使用 Lodash 库**：适用于已经在使用 Lodash 的项目。
