## 核心技术栈

### 1. **编程语言**

-   **Kotlin** - 项目主要使用 Kotlin 开发，需要熟练掌握 Kotlin 语法和特性

### 2. **框架与工具**

-   **Spring Boot** - 项目基于 Spring Boot 框架
-   **Spring MVC** - Web 层开发
-   **Spring AOP** - 切面编程（审计日志等功能）
-   **Maven** - 项目构建和依赖管理

### 3. **架构设计**

-   **DDD（领域驱动设计）** - 项目采用了 DDD 架构模式，包含：
    -   `domain` - 领域层（业务逻辑核心）
    -   `acl` - 防腐层（适配器模式）
    -   `ohs` - 应用服务层
    -   `query` - 查询服务
    -   `infrastructure` - 基础设施层
-   **六边形架构/端口适配器模式** - ports 和 adapters 目录结构

### 4. **数据库相关**

-   **SQL** - 数据库查询和设计
-   **关系型数据库** - 从`datainit.sql`可以看出使用了关系型数据库
-   **ORM 框架** - 可能使用 MyBatis 或 JPA

### 5. **中间件技术**

-   **Redis** - 缓存系统（`RedisKeys.kt`）
-   **消息队列（MQ）** - 消息发布订阅（`MQConstants.kt`、`publishers`目录）
-   **Apollo 配置中心** - 分布式配置管理

### 6. **业务领域知识**

项目涉及的业务领域：

-   数据资产管理（Data Asset）
-   数据目录（Data Catalogue）
-   数据指标（Data Indicator）
-   数据源管理（Data Source）
-   数据看板（Data Board）
-   权限管理（Auth、Role、Permission）
-   审计日志（Audit）

### 7. **其他技术点**

-   **SSE（Server-Sent Events）** - 服务器推送技术
-   **AES 加密** - 数据加密解密
-   **注解开发** - 自定义注解和 AOP
-   **Log4j2** - 日志框架

### 8. **开发工具**

-   **Git** - 版本控制
-   **IntelliJ IDEA** - Kotlin 开发推荐 IDE

## 学习建议优先级

**必须掌握（P0）：**

1. Kotlin 基础语法
2. Spring Boot 基础
3. DDD 架构理念
4. SQL 和数据库基础
5. Maven 基础

**重要掌握（P1）：**

1. Spring 核心（IoC、AOP）
2. Redis 基础使用
3. 消息队列概念
4. RESTful API 设计

**进阶掌握（P2）：**

1. 六边形架构深入理解
2. Apollo 配置中心
3. 微服务相关知识
4. 性能优化和监控

建议您先从 Kotlin 和 Spring Boot 基础开始，然后学习 DDD 架构思想，这样能更好地理解项目的整体设计。
